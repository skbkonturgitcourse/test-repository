# Домашнее задание по теме "Автоматизация"

Написать тесты на [сайт](https://qa-course.kontur.host/selenium-practice/) по подбору имени

### Перечень автоматизируемых сценариев

1. Выбрано значение "мальчик". Письмо отправляется успешно.
2. Выбрано значение "девочка". Письмо отправляется успешно.
3. После клика по ссылке "указать другой e-mail", поле для ввода e-mail доступно.
4. Не введен e-mail. Письмо не отправлено. Выводится сообщение об ошибке.
5. Введен невалидный e-mail. Письмо не отправлено. Выводится сообщение об ошибке.
